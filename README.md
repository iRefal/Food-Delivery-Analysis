I developed a Food Delivery Analysis Dashboard using simulated data to practice data analysis and visualization skills.

Key KPIs:

* Total Revenue: $2,831
* Completed Orders: $2,711
* Cancelled Orders: $120
* Average Rating: 3.67
* Completion Rate: 90%
* Average Order Value (AOV): $94.37

Analysis included:

* Revenue by restaurant
* Revenue by city
* Average delivery time by restaurant
* Order status distribution
* Rating distribution across restaurants

Tools and skills:

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
* Dashboard Design

This project is based on simulated data and was created for learning purposes. I am continuing to improve my data analysis and dashboarding skills.
